# Watchdog Docs

This folder contains project-facing collateral that is useful when presenting
or discussing Watchdog outside the codebase.

## Contents

- [Interview Questions and STAR Answers](./INTERVIEW_QUESTIONS_STAR.md)
- [Resume Bullets in X-Y-Z Format](./RESUME_BULLETS_XYZ.md)

## Intended Use

Use the interview guide when you need concise, experience-based answers about
the design, debugging, and tradeoffs in Watchdog.

Use the resume bullets when you want recruiter-friendly or hiring-manager-
friendly project descriptions that quantify the work, the implementation, and
the outcome.
